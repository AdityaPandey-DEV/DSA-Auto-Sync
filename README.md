## 🚀 Getting Started

To explore the solutions locally:

1. **Clone the repository**:
   ```bash
   git clone https://github.com/AdityaPandey-DEV/DSA-Auto-Sync.git
   ```
2. **Navigate to the directory**:
   ```bash
   cd DSA-Auto-Sync
   ```
3. **Explore specific categories** (e.g., LeetCode Medium):
   ```bash
   cd leetcode/medium
   ```
