# Contributing to DSA Auto Sync

Thank you for your interest in contributing to **DSA Auto Sync**! We welcome contributions that improve the quality of solutions, add new problems, or enhance the automation scripts.

### 1. Reporting Issues
If you find a bug, incorrect solution, or have a feature request, please [open an issue](https://github.com/AdityaPandey-DEV/DSA-Auto-Sync/issues). Include:
- A clear title and description.
- Steps to reproduce (if it's a bug).
- Platform and Problem ID (if applicable).

### 2. Submitting Changes
1. **Fork** the repository.
2. **Clone** your fork locally:
   ```bash
   git clone https://github.com/YOUR_USERNAME/DSA-Auto-Sync.git
   ```
3. **Create a new branch** for your feature or fix:
   ```bash
   git checkout -b feature/amazing-solution
   ```
4. **Commit** your changes with descriptive messages:
   ```bash
   git commit -m "Add solution for LeetCode 1234: Replace the Substring"
   ```
5. **Push** to your fork:
   ```bash
   git push origin feature/amazing-solution
   ```
6. **Open a Pull Request (PR)** to the `main` branch of this repository.
